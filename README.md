# Random Password Generator

This is a simple Random Password Generator project built using HTML, CSS, and JavaScript.  
It helps users create strong and secure random passwords.

## About the Project
This project generates random passwords using different characters like uppercase letters, lowercase letters, numbers, and special symbols.  
It is useful for creating secure passwords for accounts.

## Technologies Used
- HTML  
- CSS  
- JavaScript  

## Features
- Generates strong random passwords  
- Option to include letters, numbers, and symbols  
- Simple and user-friendly interface  
- Responsive design  

## How to Use
1. Select the password options (length, characters, symbols, etc.)  
2. Click on the "Generate Password" button  
3. Copy the generated password and use it  

## Purpose
The purpose of this project is to practice JavaScript logic and understand random functions.
